## **How to Get Started**

1. **Clone the repo**:
   ```bash
   https://github.com/himanshuladekar/FullstackPro.git
   ```
2. **Navigate to the project**:
   ```bash
   cd frontend
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```
5. **Navigate to the project**:
   ```bash
   cd backend
   ```
6. **install Dependencies**:
   ```bash
   npm init -y
   ```
